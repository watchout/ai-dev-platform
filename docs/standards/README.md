# AI Development Framework

AI駆動の開発フレームワーク。曖昧なプロダクトアイデアから実装まで、品質を保証しながら自動化する。

## 特徴

- **IEEE/ISO準拠のSSOT**: 機能仕様書を国際規格ベースで定義
- **8段階の品質ゲート**: SSOT監査からCI/PRまで、各工程で品質を保証
- **AI中断プロトコル**: AIが不明点を推測せず、必ず確認する体制
- **完全な開発ライフサイクル**: アイデア → 仕様 → 実装 → テスト → デプロイ → 保守

## 使い方

### 新規プロジェクト

1. "Use this template" でリポジトリを作成
2. Claude Code で `CLAUDE.md` を読ませる
3. ディスカバリーから開発開始

### 既存プロジェクト

1. `docs/standards/` をプロジェクトにコピー
2. 既存コードから機能を逆引きでSSOT化
3. 通常フローに合流
